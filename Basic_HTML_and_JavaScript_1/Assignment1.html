Styling with HTML

In html we can styalize font by adding the following elements:

To make text bold you would use the <b> element.

To italicize text you would use the <i> element.

To underline text you would use the <u> element.

To change font style you would add the face attribute.

For example: <font face="Arial">Your text here.</font>
